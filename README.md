Introduction to Python

Python is a widely used open-source programming language known for its simplicity and readability. It is highly versatile and used in various fields such as web development, data science, machine learning, and automation.

⸻

Project Overview

This project focuses on analyzing Python as an open-source software from both theoretical and practical perspectives. The key areas covered include:
	•	Study of Python’s origin and philosophy
	•	Analysis of its open-source license
	•	Understanding its role in Linux-based systems
	•	Exploration of the open-source ecosystem
	•	Comparison with proprietary software
	•	Implementation of shell scripts to demonstrate Linux concepts

⸻

Shell Scripts Description

Script 1 — System Identity Report
Displays essential system details such as kernel version, current user, system uptime, date, and Linux distribution.

Script 2 — FOSS Package Inspector
Checks whether Python is installed on the system and shows its details along with a brief description using a case statement.

Script 3 — Disk and Permission Auditor
Examines important system directories and displays their size, permissions, owner, and group information.

Script 4 — Log File Analyzer
Reads a log file, counts occurrences of a specific keyword (default: “error”), and shows the most recent matching entries.

Script 5 — Open Source Manifesto Generator
Takes user input and generates a personalized open-source manifesto, which is saved as a text file.


Dependencies

	•	Bash Shell
	•	dpkg (for package management)
	•	Standard Linux utilities: grep, awk, du, ls, cut
